Este es un proyecto ambicioso y muy necesario para una institución educativa. Dado que el presupuesto es limitado pero se cuenta con voluntad técnica y hardware disponible, el enfoque debe ser escalable, modular y basado en software de código abierto.
Aquí tienes una propuesta estructural, técnica y funcional para el desarrollo de la aplicación, que llamaremos provisionalmente "SmartPark Tech".
1. Arquitectura Tecnológica (Stack Recomendado)
Para lograr ese aspecto "Premium", moderno y que funcione tanto en escritorio como en móvil sin costo de licencias:
Frontend: React.js o Next.js con Tailwind CSS. Esto permite crear interfaces ultra modernas, con modo oscuro nativo y una estética "limpia".
Backend: Node.js (Express) o Python (FastAPI). Si planeas integrar reconocimiento de placas (OCR), Python es la mejor opción por sus librerías de Inteligencia Artificial.
Base de Datos: PostgreSQL. Es robusta, gratuita y maneja muy bien relaciones complejas (usuarios, vehículos, registros, centros).
Despliegue: Inicialmente en un servidor local de la institución para evitar costos de nube, accesible vía IP local o VPN.
2. Módulos y Funcionalidades Clave
A. Gestión Dinámica de Inventario (Flex-Grid)
En lugar de una tabla estática, usaremos un Mapa Lógico de Parqueaderos.
Configurador de Zonas: Una interfaz donde el admin puede crear "Bloques" (Ej: Bloque A - Centro de Electricidad).
Tipado Dinámico: Cada celda o cupo puede ser editado para cambiar de "General" a "Directivo" o "Discapacitado" con un clic.
Contador en Tiempo Real: Un dashboard que reste automáticamente los ingresos de la capacidad total por tipo de vehículo.
B. Registro de Usuarios y "Centros"
Perfil Multicapa: Nombre, ID, Rol (Estudiante/Inst/Admin/Visitante), Centro (Centro 1, 2, 3 u Otros).
Vinculación de Vehículos: Un usuario puede tener registrados varios vehículos (una moto y un carro), pero solo uno puede estar activo en el sistema a la vez.
C. Control de Acceso y OCR (Automatización Low-Cost)
Para automatizar sin presupuesto:
Sugerencia OCR: Utilizar una cámara web o de seguridad (IP) apuntando a la entrada. Usar la librería EasyOCR o Tesseract (gratuitas en Python).
Flujo: La cámara detecta la placa -> El sistema busca en la DB -> Si existe, muestra los datos al vigilante en pantalla -> El vigilante solo presiona "Enter" para confirmar ingreso (ya que la talanquera es manual). Esto ahorra tiempo de digitación.
D. Caracterización y Alertas (Sistema de "Flags")
Hoja de Vida del Conductor: El sistema registrará incidentes.
Alerta Amarilla: Velocidad excedida o mal parqueado (el vigilante lo reporta en la app).
Alerta Roja: Reincidencia o parqueo en zona de discapacitados sin permiso.
Notificaciones: Al ingresar, si el usuario tiene una nota pendiente, al vigilante le sale un pop-up: "Informar al usuario que debe parquear en reversa".
3. Interfaz de Usuario (UI/UX Premium)
Paleta de Colores: Fondo Slate-900 para modo oscuro, con acentos en Emerald-500 (disponible) y Rose-500 (ocupado). Tipografía tipo Inter o Roboto para legibilidad técnica.
Dashboard de Portería: Una vista simplificada con botones gigantes para uso rápido en pantallas táctiles o con teclado numérico.
Vista Pública (PWA): Una URL simple (ej: parqueadero.inst.edu/estado) donde cualquier estudiante vea un velocímetro o barra de progreso: "Motos: 85% lleno".
4. Ideas Extra para Profesionalismo "Top"
Generación de Códigos QR para Visitantes: Cuando alguien externo va a ingresar, el funcionario que lo invita genera un QR desde la app, el visitante lo muestra en portería y el registro es instantáneo.
Mapa de Calor: Una sección para la administración que muestre qué días y a qué horas colapsa el parqueadero (ideal para toma de decisiones de infraestructura).
Auditoría de Roles: Registro de qué vigilante o funcionario autorizó cada entrada/salida para evitar ingresos no autorizados.
Auto-Registro (Onboarding): Un módulo donde el estudiante sube su tarjeta de propiedad y documento, y un administrador solo da "Aprobar". Esto evita cargar los datos manualmente.

import React, { useState, useMemo, useEffect } from 'react';
import { 
  LayoutDashboard, Car, Bike, UserCheck, ShieldAlert, 
  Moon, Sun, Search, LogIn, LogOut, Laptop, Wrench, 
  Map as MapIcon, ClipboardList, FileBarChart, Settings, 
  UserPlus, Clock, Building2, User
} from 'lucide-react';

// --- MOCK DE BASE DE DATOS INICIAL (USUARIOS REGISTRADOS) ---
const MOCK_DB_USUARIOS = [
  { plate: 'ABC123', name: 'Juan Pérez', type: 'Instructor', center: 'CDTI', assets: ['Portátil Asus', 'Caja de Herramientas'] },
  { plate: 'MOT999', name: 'Maria Gomez', type: 'Estudiante', center: 'CBA', assets: ['Casco'] },
];

const SmartParkApp = () => {
  const [darkMode, setDarkMode] = useState(true);
  const [activeTab, setActiveTab] = useState('dashboard');
  
  // Estados de Registro
  const [plateInput, setPlateInput] = useState('');
  const [isVisitor, setIsVisitor] = useState(false);
  const [tempUser, setTempUser] = useState({ name: '', type: 'Estudiante', center: 'CDTI', vehicleType: 'Carro', hasAssets: false });
  
  // Estado de Vehículos Adentro
  const [logs, setLogs] = useState([]);
  const [history, setHistory] = useState([]); // Para ver las salidas

  // --- LÓGICA DE DETECCIÓN DE USUARIO ---
  useEffect(() => {
    const found = MOCK_DB_USUARIOS.find(u => u.plate === plateInput);
    if (found) {
      setTempUser({ ...found, vehicleType: plateInput.length > 3 ? 'Carro' : 'Moto', hasAssets: found.assets.length > 0 });
      setIsVisitor(false);
    } else if (plateInput.length > 3) {
      setIsVisitor(true);
    }
  }, [plateInput]);

  // --- ESTADÍSTICAS ---
  const stats = useMemo(() => ({
    Carros: { occupied: logs.filter(l => l.vehicleType === 'Carro').length, total: 50 },
    Motos: { occupied: logs.filter(l => l.vehicleType === 'Moto').length, total: 120 },
    Bicis: { occupied: logs.filter(l => l.vehicleType === 'Bici').length, total: 30 },
    Visitantes: { occupied: logs.filter(l => l.type === 'Visitante').length, total: 20 }
  }), [logs]);

  // --- ACCIÓN: REGISTRAR ENTRADA ---
  const handleRegister = () => {
    if (!plateInput) return;
    if (logs.find(l => l.plate === plateInput)) {
      alert("EL VEHÍCULO YA ESTÁ ADENTRO");
      return;
    }

    const newEntry = {
      id: Date.now(),
      plate: plateInput.toUpperCase(),
      user: isVisitor ? tempUser.name : tempUser.name,
      type: isVisitor ? 'Visitante' : tempUser.type,
      center: tempUser.center,
      entryTime: new Date(),
      vehicleType: tempUser.vehicleType,
      hasAssets: tempUser.hasAssets
    };

    setLogs([newEntry, ...logs]);
    setPlateInput('');
    setTempUser({ name: '', type: 'Estudiante', center: 'CDTI', vehicleType: 'Carro', hasAssets: false });
  };

  // --- ACCIÓN: REGISTRAR SALIDA ---
  const handleExit = (id) => {
    const logToExit = logs.find(l => l.id === id);
    const exitTime = new Date();
    const duration = Math.round((exitTime - logToExit.entryTime) / 60000); // minutos

    setHistory([{ ...logToExit, exitTime, duration }, ...history]);
    setLogs(logs.filter(l => l.id !== id));
  };

  return (
    <div className={`${darkMode ? 'bg-slate-950 text-slate-100' : 'bg-gray-50 text-slate-900'} min-h-screen transition-colors duration-300 font-sans`}>
      
      {/* SIDEBAR (Versión Premium) */}
      <aside className={`fixed left-0 top-0 h-full w-64 ${darkMode ? 'bg-slate-900 border-slate-800' : 'bg-white border-gray-200'} border-r z-20`}>
        <div className="p-6">
          <h1 className="text-2xl font-black bg-gradient-to-r from-emerald-400 to-blue-500 bg-clip-text text-transparent italic tracking-tighter">SENA PARK</h1>
          <p className="text-[10px] text-slate-500 font-bold uppercase tracking-widest">Control Técnico Institucional</p>
        </div>
        <nav className="mt-6 px-4 space-y-1">
          {[
            { id: 'dashboard', label: 'Portería', icon: <LayoutDashboard size={20}/> },
            { id: 'map', label: 'Mapa General', icon: <MapIcon size={20}/> },
            { id: 'users', label: 'Usuarios / Roles', icon: <UserCheck size={20}/> },
            { id: 'assets', label: 'Activos / Equipos', icon: <Laptop size={20}/> },
            { id: 'incidents', label: 'Incidentes', icon: <ShieldAlert size={20}/> },
            { id: 'reports', label: 'Reportes', icon: <FileBarChart size={20}/> },
            { id: 'settings', label: 'Configuración', icon: <Settings size={20}/> },
          ].map((item) => (
            <button key={item.id} onClick={() => setActiveTab(item.id)}
              className={`w-full flex items-center space-x-3 px-4 py-3 rounded-xl transition-all ${activeTab === item.id ? 'bg-emerald-500/10 text-emerald-500 border border-emerald-500/20 font-bold' : 'text-slate-400 hover:bg-slate-800 hover:text-slate-200'}`}>
              {item.icon} <span className="text-sm">{item.label}</span>
            </button>
          ))}
        </nav>
      </aside>

      {/* CONTENIDO PRINCIPAL */}
      <main className="ml-64 p-8">
        <header className="flex justify-between items-center mb-8">
          <div>
            <h2 className="text-3xl font-black tracking-tight">Panel de Portería</h2>
            <div className="flex items-center text-xs font-bold text-slate-500 uppercase gap-2">
              <Clock size={14}/> {new Date().toLocaleDateString()} | Horario Diurno
            </div>
          </div>
          <button onClick={() => setDarkMode(!darkMode)} className={`p-3 rounded-full ${darkMode ? 'bg-slate-800 text-yellow-400' : 'bg-white shadow-md'}`}>
            {darkMode ? <Sun size={20}/> : <Moon size={20}/>}
          </button>
        </header>

        {activeTab === 'dashboard' && (
          <div className="space-y-8">
            {/* TARJETAS DE ESTADO (Iconos Grandes) */}
            <div className="grid grid-cols-1 md:grid-cols-4 gap-6">
              {Object.entries(stats).map(([label, data], idx) => (
                <div key={idx} className={`${darkMode ? 'bg-slate-900 border-slate-800' : 'bg-white border-gray-100 shadow-sm'} p-6 rounded-2xl border`}>
                  <div className="flex justify-between items-start mb-4">
                    <div className="p-3 bg-emerald-500/10 text-emerald-500 rounded-xl">
                      {label === 'Carros' ? <Car size={40}/> : label === 'Motos' ? <Bike size={40}/> : <User size={40}/>}
                    </div>
                    <span className="text-[10px] font-black text-slate-500 uppercase">{label}</span>
                  </div>
                  <div className="flex items-baseline gap-2">
                    <h3 className="text-5xl font-black">{data.occupied}</h3>
                    <span className="text-xl text-slate-500 font-bold">/ {data.total}</span>
                  </div>
                </div>
              ))}
            </div>

            <div className="grid grid-cols-1 lg:grid-cols-3 gap-8">
              {/* PANEL DE REGISTRO DINÁMICO */}
              <div className={`${darkMode ? 'bg-slate-900 border-emerald-500/20' : 'bg-white shadow-2xl'} p-8 rounded-3xl border-2`}>
                <h3 className="text-xl font-bold mb-6 flex items-center gap-2">
                  <LogIn className="text-emerald-500" /> Registro de Entrada
                </h3>
                
                <div className="space-y-5">
                  {/* Entrada de Placa */}
                  <div>
                    <label className="text-[10px] font-black text-slate-500 uppercase tracking-widest">Placa del Vehículo</label>
                    <input type="text" value={plateInput} onChange={(e) => setPlateInput(e.target.value.toUpperCase())}
                      placeholder="ABC-123" className={`w-full mt-1 p-5 rounded-2xl border-2 ${darkMode ? 'bg-slate-800 border-slate-700' : 'bg-gray-50 border-gray-200'} focus:border-emerald-500 outline-none text-4xl font-black text-center tracking-tighter`} />
                  </div>

                  {/* Datos Autocompletados o Visitante */}
                  <div className={`p-4 rounded-2xl ${isVisitor ? 'bg-orange-500/5 border border-orange-500/20' : 'bg-emerald-500/5 border border-emerald-500/20'}`}>
                    <div className="flex justify-between items-center mb-3">
                      <span className="text-[10px] font-bold uppercase">{isVisitor ? '⚠️ Modo Visitante' : '✅ Usuario Reconocido'}</span>
                      {isVisitor && <UserPlus size={16} className="text-orange-500"/>}
                    </div>
                    <input type="text" placeholder="Nombre Completo" value={tempUser.name} onChange={(e) => setTempUser({...tempUser, name: e.target.value})}
                      className={`w-full mb-2 p-3 rounded-lg text-sm ${darkMode ? 'bg-slate-800' : 'bg-white border'}`} disabled={!isVisitor} />
                    
                    <div className="grid grid-cols-2 gap-2">
                      <select value={tempUser.type} onChange={(e) => setTempUser({...tempUser, type: e.target.value})} className={`p-2 rounded-lg text-xs ${darkMode ? 'bg-slate-800' : 'bg-white border'}`} disabled={!isVisitor}>
                        <option>Estudiante</option><option>Instructor</option><option>Administrativo</option><option>Visitante</option>
                      </select>
                      <select value={tempUser.center} onChange={(e) => setTempUser({...tempUser, center: e.target.value})} className={`p-2 rounded-lg text-xs ${darkMode ? 'bg-slate-800' : 'bg-white border'}`}>
                        <option>CDTI</option><option>CBA</option><option>ASTIN</option><option>OTRO</option>
                      </select>
                    </div>
                  </div>

                  {/* Selector de Vehículo */}
                  <div className="flex gap-2">
                    {['Carro', 'Moto', 'Bici'].map(v => (
                      <button key={v} onClick={() => setTempUser({...tempUser, vehicleType: v})}
                        className={`flex-1 py-3 rounded-xl border-2 transition-all font-bold text-xs ${tempUser.vehicleType === v ? 'border-emerald-500 bg-emerald-500/10 text-emerald-500' : 'border-slate-800 text-slate-500'}`}>
                        {v}
                      </button>
                    ))}
                  </div>

                  {/* Checklist de Activos */}
                  <label className="flex items-center gap-3 p-4 rounded-2xl border border-slate-800 cursor-pointer hover:bg-slate-800/30 transition-all">
                    <input type="checkbox" checked={tempUser.hasAssets} onChange={(e) => setTempUser({...tempUser, hasAssets: e.target.checked})} className="w-5 h-5 accent-emerald-500" />
                    <div className="flex items-center gap-2">
                      <Laptop size={18} className="text-slate-400"/>
                      <span className="text-xs font-bold uppercase">Ingresa con Equipos/Activos</span>
                    </div>
                  </label>

                  <button onClick={handleRegister} className="w-full py-5 bg-emerald-600 hover:bg-emerald-500 text-white rounded-2xl text-xl font-black shadow-lg shadow-emerald-900/40 transition-all active:scale-95">
                    CONFIRMAR INGRESO
                  </button>
                </div>
              </div>

              {/* LISTA DE VEHICULOS EN PLANTA */}
              <div className={`lg:col-span-2 ${darkMode ? 'bg-slate-900 border-slate-800' : 'bg-white border-gray-100 shadow-sm'} p-8 rounded-3xl border`}>
                <div className="flex justify-between items-center mb-6">
                  <h3 className="text-xl font-bold italic">Vehículos en Planta</h3>
                  <div className="flex items-center gap-2 text-emerald-500 text-[10px] font-black uppercase tracking-tighter">
                    <div className="w-2 h-2 bg-emerald-500 rounded-full animate-pulse"></div>
                    Monitoreo en Tiempo Real
                  </div>
                </div>

                <div className="overflow-hidden">
                  <table className="w-full text-left">
                    <thead>
                      <tr className="border-b border-slate-800 text-slate-500 text-[10px] font-black uppercase tracking-widest">
                        <th className="pb-4">Vehículo / Placa</th>
                        <th className="pb-4">Propietario / Centro</th>
                        <th className="pb-4 text-center">Activos</th>
                        <th className="pb-4 text-right">Ingreso</th>
                        <th className="pb-4 text-right">Acción</th>
                      </tr>
                    </thead>
                    <tbody className="divide-y divide-slate-800/50">
                      {logs.map((log) => (
                        <tr key={log.id} className="group hover:bg-slate-800/20 transition-all">
                          <td className="py-5">
                            <span className="text-4xl font-black font-mono text-emerald-400 tracking-tighter block">{log.plate}</span>
                            <span className="text-[10px] font-bold text-slate-500 uppercase tracking-widest">{log.vehicleType}</span>
                          </td>
                          <td>
                            <div className="flex flex-col">
                              <span className="text-lg font-bold">{log.user}</span>
                              <span className="text-[10px] font-black text-slate-500 uppercase">{log.type} | {log.center}</span>
                            </div>
                          </td>
                          <td className="text-center">
                            {log.hasAssets ? <div className="inline-flex p-2 bg-blue-500/10 text-blue-500 rounded-lg"><Laptop size={16}/></div> : <span className="text-slate-700">-</span>}
                          </td>
                          <td className="text-right text-sm font-mono font-bold text-slate-400">
                            {log.entryTime.toLocaleTimeString([], { hour: '2-digit', minute: '2-digit' })}
                          </td>
                          <td className="text-right">
                            <button onClick={() => handleExit(log.id)} className="p-4 bg-rose-500/10 text-rose-500 rounded-2xl hover:bg-rose-500 hover:text-white transition-all shadow-sm">
                              <LogOut size={24}/>
                            </button>
                          </td>
                        </tr>
                      ))}
                    </tbody>
                  </table>
                </div>
              </div>
            </div>
          </div>
        )}

        {activeTab !== 'dashboard' && (
          <div className="h-96 flex flex-col items-center justify-center border-2 border-dashed border-slate-800 rounded-3xl animate-pulse">
            <Settings size={64} className="text-slate-800 mb-4" />
            <h3 className="text-xl font-bold text-slate-500 uppercase tracking-widest">Módulo {activeTab}</h3>
            <p className="text-sm text-slate-600 uppercase font-bold tracking-tighter">Preparando Estructura de Datos...</p>
          </div>
        )}
      </main>
    </div>
  );
};

export default SmartParkApp;

Lógica de Tiempo: Se registra la hora exacta de entrada (entryTime). Al dar clic en el botón rojo de salida, el vehículo se elimina de "En Planta" (puedes ver la lógica para que esto alimente un historial de reportes luego).
